Java 刷题必补基础总结

1️⃣ List / ArrayList

关键技术：顺序容器、动态数组
总结：
List 是接口，不能直接 new，常用实现是 ArrayList。
刷题中用于存放结果集，常用 add / get / size / remove，支持下标访问，时间复杂度通常为 O(1)（尾部操作）。

2️⃣ Map / HashMap

关键技术：键值映射、哈希表
总结：
Map 用于“分类 + 快速查找”，常见用法是 key 表示特征，value 存放结果集合。
刷题中常用 get / put / containsKey / putIfAbsent / values，时间复杂度通常为 O(1)。

3️⃣ Set / HashSet

关键技术：去重、哈希表
总结：
Set 天然不允许重复元素，适合用于判重和快速查找。常与滑动窗口、哈希思想结合使用，时间复杂度 O(1)。

4️⃣ String / char[] 转换

关键技术：字符串与字符数组互转
总结：
String 不可变，字符操作常转为 char[] 处理。
刷题中常用 toCharArray()、new String(char[])、charAt() 来进行字符级操作。

5️⃣ Arrays 工具类

关键技术：数组排序与比较
总结：
Arrays 提供常用数组操作，如 Arrays.sort() 和 Arrays.equals()。
常用于排序、字母异位词、双指针等场景。
